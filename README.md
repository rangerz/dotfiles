# rangerz/dotfiles

My dotfiles, such as bash alias, git, vim, and ssh setting.

### Install Config

```bash
# Install
cd ~
git clone https://github.com/rangerz/dotfiles.git .dotfiles
~/.dotfiles/install.sh

# Uninstall
~/.dotfiles/uninstall.sh
```

### Features

- shell alias
- git config
- vim config
- macos config
