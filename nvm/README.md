# NVM (Node Version Manager)

```bash
# 安装指定版本的 Node.ts
nvm install [version]

# 使用指定版本的 Node.ts
nvm use [version]

# 卸载指定版本的 Node.ts
nvm uninstall [version]

# 列出已安装的所有 Node.ts 版本
nvm list

# 查看 nvm 的根目录路径
nvm root

# 开启 Node.ts 版本管理
nvm on

# 关闭 Node.ts 版本管理
nvm off

# 设置下载代理
nvm proxy [url]

# 设置或者查看 setting.txt 中的 node_mirror
nvm node_mirror [url]

# 设置或者查看 setting.txt 中的 npm_mirror
nvm npm_mirror [url]

# 设置默认 node 版本
nvm alias default <version>
```
